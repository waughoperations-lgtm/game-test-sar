# Hide and Seek Town

A 3D hide-and-seek game you play in your web browser. It's nighttime — grab your
flashlight and find Pip, Momo and Zuzu before the 2-minute clock runs out!

## How to play it right now
Just double-click `index.html` and it opens in your browser. Nothing to install.

- Computer: walk with W A S D or the arrow keys, click-and-drag to look around
- Phone/tablet: slide your left thumb to walk, right thumb to look around

## How to keep building it with Claude Code
1. Put this whole folder somewhere easy to find (like your Documents folder)
2. Open Claude Code and open this folder
3. Just describe what you want in plain English, for example:
   - "Add giggle sounds that get louder when I'm close to a hider"
   - "Make the hiders sneak to a new spot if I take too long"
   - "Add a third difficulty with 5 hiders and only 90 seconds"
4. Claude Code will make the changes — then double-click index.html to try them

The CLAUDE.md file in this folder tells Claude Code everything about the project,
so you can just start asking for changes.
